# Roadmap

NexusClaw is in an early open-source release. The current focus is making the project easier to understand, run, evaluate, and extend.

## Near Term

- Add screenshots for the admin console, portal chat, file upload flow, and source-aware answers
- Add a short demo GIF that shows the first-run flow from sample document upload to answer inspection
- Improve the demo project with expected API/UI walkthroughs
- Add a `docker-compose.demo.yml` profile for a more complete local demo stack
- Document a first successful answer flow using `examples/demo-project`
- Add starter issues for documentation, demo data, and local setup improvements

## Engineering Priorities

- Strengthen project-level authorization and RBAC
- Improve snippet-level citations across all chat paths
- Expand vector search backend options and local defaults
- Improve deployment documentation and production hardening notes
- Add repeatable evaluation examples and benchmark fixtures

## Longer Term

- External source connectors
- More scene workflow examples
- More admin governance workflows
- Hosted demo environment
- Versioned releases with upgrade notes
