# frontend/portal-web

## 目标

提供面向市民和工作人员的 NexusClaw 问答前台。

## 核心页面

- 首页 / 欢迎页
- 对话页
- 历史会话页

## 核心能力

- 展示项目级开场语
- 展示推荐问题
- 发起多轮问答
- 展示来源知识
- 查看历史对话

## 建议技术栈

- React
- Umi
- TypeScript
- Ant Design
